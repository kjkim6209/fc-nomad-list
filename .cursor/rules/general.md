## 언어 & 커뮤니케이션

- **한국어 우선**
  - AI가 생성하는 모든 자연어 설명, 주석, README, 커밋 메시지 제안 등은 **반드시 한국어**로 작성한다.

## 경로 & 디렉터리 사용

- **경로 표기**
  - 툴 호출 인자나 설명에서 가능한 경우 **절대 경로**를 선호한다.
  - 코드 내 import 문 등 실제 코드에서는 일반적인 상대 경로 규칙을 따른다.

## 코드 스타일 & 구조

- **코드 스타일**
  - TypeScript / React / Next.js의 최신 문법을 사용한다.
  - 불필요한 축약보다는 **명확한 이름**을 우선한다.
  - 유틸 함수는 `lib/` 또는 `hooks/`에, UI는 `components/`에 배치한다.

## UI/UX & 성능

- **UX / UI**
  - 이 프로젝트는 **모던하고 깔끔한 UI/UX**를 지향한다.
  - 가능하면 기존 UI 컴포넌트(`components/ui`)를 재사용하고, 디자인 일관성을 유지한다.

- **성능 / 접근성**
  - 가능한 경우 React/Next의 **서버 컴포넌트**/클라이언트 컴포넌트 구분을 명확히 한다.
  - 이미지, 리스트 등에서는 성능과 접근성을 고려한다 (예: `next/image`, 적절한 aria 속성).
